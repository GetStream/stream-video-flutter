# OutgoingCallContent

The `OutgoingCallContent` widget displays the outgoing call which includes information about the call
and options to cancel it. The widget can be used on its own or as a part of `StreamCallContainer`.

## Usage

```dart
    OutgoingCallContent(
      call: call,
      callState: callState,
      onCancelCallTap: () {},
    ),
```
