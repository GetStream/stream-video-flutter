# IncomingCallContent

The `IncomingCallContent` widget displays the incoming call which includes information about the call
and options to accept and decline it. The widget can be used on its own or as a part of `StreamCallContainer`.

## Usage

```dart
    IncomingCallContent(
      call: call,
      callState: callState,
      onAcceptCallTap: () {},
      onDeclineCallTap: () {},
    ),
```
