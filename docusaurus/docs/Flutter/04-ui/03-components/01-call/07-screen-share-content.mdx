# ScreenShareContent

The `ScreenShareContent` widget renders a screen-sharing track on the screen.

## Usage

To use the `ScreenShareContent` widget, pass in the call and call state to the widget.

```dart
    ScreenShareContent(
      call: call,
      participant: callState,
    ),
```

## Customisation

You can customise the background color using the `backgroundColor` parameter.

For example, for a `ScreenShareContent` with a blue background color:

```dart
    ScreenShareContent(
      call: call,
      participant: callState,
      backgroundColor: Colors.blue,
    ),
```
